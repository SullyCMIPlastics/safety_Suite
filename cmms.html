Import Assets from FIIX CSV Export
How to export your assets from FIIX:
  1. In FIIX, go to Assets → Equipment
  2. Click the Reports button (or the grid/list export icon)
  3. Choose "Export to CSV" or "Export to Excel (.xlsx)"
  4. Save the file to your computer
  5. For attachments: open each asset in FIIX → Files tab → download each file manually, then re-upload them here after import
Supported columns (auto-detected): Name, Code/Tag, Category, Status, Location, Make/Manufacturer, Model, Serial Number, Description, Notes, Install Date, Cost
